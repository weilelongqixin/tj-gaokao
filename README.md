# 天津高考志愿填报助手

一个面向**天津高考考生家长**的纯静态 Web 工具。输入孩子的高考分数和选考科目，即可：

1. **换算全市位次** — 孩子这个分数在全市排第几名
2. **等位分换算** — 同样的位次，在往年（2023/2024）相当于多少分（位次法比直接比分更准）
3. **冲稳保报考范围** — 结合位次和选科，筛出可报的院校专业组，按 冲/稳/保 三档分组

> ⚠️ **重要声明**：本工具是**参考性辅助工具**，不构成填报建议的最终依据。院校录取数据、招生计划、选科要求每年都会调整，最终填报务必以**天津市教育招生考试院「志愿填报辅助系统」**及各高校招生章程为准。

## 为什么做这个

天津是 3+3 新高考，按"院校专业组"投档，每个专业组有独立的录取线和选科要求。官方的"志愿填报辅助系统"功能强大但对家长不够直观。这个工具聚焦它不强调的环节——**用位次法把孩子的分数翻译成"往年能上什么层次"，并给出一份冲稳保的范围地图**。

## 如何使用（本地预览）

由于浏览器对本地 `file://` 加载 JSON 有跨域限制，需用一个本地服务器：

```bash
cd tj-gaokao
# 方式一：Python 内置服务器
python3 -m http.server 8000
# 然后浏览器打开 http://localhost:8000

# 方式二：Node 的 serve
npx serve .
```

## 核心算法说明

### 位次法（比直接比分靠谱）
每年试卷难度、考生人数都在变，直接拿今年分数去比往年分数误差很大。正确做法是：
- 今年分数 → 查今年一分一段表 → 得到**全市位次**（相对水平）
- 把这个位次放到往年一分一段表里反查 → 得到**往年等位分**

位次反映了考生在全市的相对位置，跨年可比。

### 冲稳保分档（基于位次相对差）
```
ratio = (专业组往年录取位次 − 考生位次) / 考生位次
```
- ratio < 0：专业组分数高于考生 → **冲**（录取位次越小越难够到）
- ratio ≈ 0：位次接近 → **稳**
- ratio > 0：专业组分数低于考生 → **保**（留有余量）

具体阈值见 `js/matcher.js`。

## 更新数据

### 更新一分一段表
每年新表公布后（通常 6 月下旬），从教育在线天津专题页抓取，替换 `data/score_rank/tianjin_YYYY.json`，并在 `js/app.js` 的 `BASE_YEAR`、`COMPARE_YEARS`、`LINE_SCORES` 中更新年份。

### 更新院校数据
运行 `python3 tools/fetch_colleges_full.py`（全量抓取，带断点续传，约 45-75 分钟）和 `python3 tools/transform_colleges.py`（转换格式）。中途中断后重跑会自动跳过已抓院校。

## 已知局限

- **院校范围**：已全量覆盖全国所有在天津有本科招生的院校（805 所）。仍有少数院校（新设院校、当年新增招生计划）可能缺失。
- **选科要求版本**：选科要求来自抓取时的数据版本，可能与最新招生目录有出入。
- **无招生计划**：本工具不展示各专业组招生人数，需到官方系统查看。
- **历史数据深度**：院校录取数据为 2023/2024 两年，2025 录取数据待官方公布后补充。
